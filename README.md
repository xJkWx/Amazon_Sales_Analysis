Amazon Sales Analysis — SQL Project

## Overview
Exploratory analysis of Amazon e-commerce sales data using SQL.  
The goal was to identify key sales trends, top-performing categories, 
and customer segments to support business decision-making.


## Business Questions Answered

1. Which product categories generate the highest revenue?
2. What is the distribution of orders by status?
3. Which cities have the highest sales volume?
4. How does B2B compare to B2C in terms of revenue and order value?
5. Which sales channel performs better?

## Key Findings

- Top 3 categories (Set, Kurta, Western Dress) account for 89% of all orders
- B2B orders have 8% higher average order value than B2C
- City BENGALURU leads with 9,905 orders"
